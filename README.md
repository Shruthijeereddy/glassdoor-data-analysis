# Glassdoor Company Analysis Dashboard

## Overview
This project analyzes Glassdoor company data to explore company ratings, industries, salaries, and employee insights using Python and interactive visualizations.

## Objective
The goal of this project is to perform exploratory data analysis (EDA) and build an interactive dashboard to help users explore company information and identify top-rated organizations.

## Tools & Technologies
- Python
- Pandas
- Streamlit
- Plotly
- Data Visualization

## Features
- User login and profile system
- Interactive company recommendations
- Industry and location filtering
- Top company ranking analysis
- Visual charts and comparisons

## Dataset
Dataset sourced from Kaggle Glassdoor company dataset.

## Key Learning Outcomes
- Data cleaning and preprocessing
- Exploratory data analysis
- Dashboard development using Streamlit
- Building interactive data applications
